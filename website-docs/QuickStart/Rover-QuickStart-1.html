1. Installation

Rover installs like any other perl module. However, it is not avilable through CPAN, so it must be installed manually. Follow these steps in order to get Rover working on your system.

1.1 Dependencies

There is only one hard dependency for Rover, and that is Expect for perl. The easiest way to get this installed (if you do not already have it), is to let The Rover installation script install it for you.

If you already have Expect, then you should be able to install at least the Rover core package and the command line interface. Rover is 100% Perl, and requires no C compiler to install.

If, however, you would like to run the graphical user interface, you need to also have the Gtk perl module. Again, the easiest way to get this is to let the Rover installation script install it for you.

Note: If you chose to let the installer build Gtk-Perl, you must have the gtk+ development packages installed. Please refer to your OS documentation for details on installing these.

1.2 Download

In order to download the latest copy of Rover, use the download section of the SourceForge project page.

The file should be a gziped tar file named Rover-2.xx.tar.gz. The xx denotes the latest version.

1.3 Build and Install

After downloading Rover, extract the contents using the following syntax:

$ gunzip -c Rover-2.xx.tar.gz | tar xvf -
After it is extracted, cd into the Rover-2.xx directory and install it as a normal Perl module. The steps for this are:
$ perl Makefile.PL
$ make
$ su
# make install
Alternatively you can chose to have the installation script build and install the dependancies for you. The install script will use CPAN to install missing perl modules. If you have not used CPAN before, you will be prompted to configure it. Here is the syntax for using this option:
$ perl Makefile.PL --install-deps
If you chose not to install the graphical interface, but still wish to use the command line, use this command instead:
$ perl Makefile.PL --no-gtk
The above two examples will require an additional "make" and "make install".
